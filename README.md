### Installation

#### 1. Requirements
* Metamask Extension
* Coinbase Wallet Extension

#### 2. Clone this repo
```bash
 git clone https://github.com/zackyfachrur/stick-chain-web3.git
```

#### 3. Install stick-chain-web3
```bash
 cd stick-chain-web3
 npm install 
```

### Tech Stack

**Client:** React, Typescript, TailwindCSS
